# My Neovim Cheetsheet

## Keybindings

| keybinding | description                    |
| ---------- | ------------------------------ |
| `C-i`      | move forward in jump history   |
| `C-o`      | move backwards in jump history |

### Telescope

| keybinding | description                      |
| ---------- | -------------------------------- |
| `<C-x>`    | go to file selection as a split  |
| `<C-v>`    | go to file selection as a vsplit |
